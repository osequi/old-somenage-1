## useLinearScale(value, linearScale) â‡’
Returns a value from the linear scale.

**Kind**: global function  
**Returns**: The value from the scale.  
**Category**: Hooks  

| Param | Description |
| --- | --- |
| value | The value on the scale. |
| linearScale | The settings for the scale. |

**Example**  
```js
useLinearScale(0) => 1
useLinearScale(1) => 2
```
