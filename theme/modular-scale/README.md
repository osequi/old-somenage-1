## modularScale
Defines the modular scale.

On modular scale values scale based on the settings. See 'useModularScale'.

**Kind**: global variable  
**Category**: Theme  
**See**: https://github.com/modularscale/modularscale-js  
**Example**  
```js
name: 'modular',
settings: {base:[1], ratio: 1.33}
```
