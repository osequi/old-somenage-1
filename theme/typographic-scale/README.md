## typographicScale
Defines the typographic scale.

The scale comes from one of the presets: linear scale, modular scale.

**Kind**: global variable  
**Category**: Theme  
**Example**  
```js
name: "linear",
settings: {},
```
**Example**  
```js
name: 'modular',
settings: {base:[1], ratio: 1.33}
```
